Projects

Sci-fi themed project logs adapted from my other site.

Project 1 - Floating Platforms

A 2D movement game about jumping across floating platforms. The focus is crisp controls, readable jumps, and rearrangeable level chunks for harder routes.

Project 2 - 3D Time Trials

A 3D platformer built around speed and time trials, with tight routes, shortcuts, and quick restarts to improve run consistency and timer flow.

Project 3 - Speed and Precision

A 2D platformer for players who like speed, precision, and time-based puzzle pressure, including challenge paths that reward cleaner execution.

Project 4 - Break the Game

A puzzle game concept where intended exploits and weird interactions are the intended way to solve levels instead of something to patch out.

WIP

Current work-in-progress concepts adapted into the sci-fi style.

WIP 1 - Frisbee Arena

A frisbee-style game where the disc can bounce off walls very hard, creating chaotic angles and fast reactions. The current focus is tuning physics, camera readability, and scoring flow.

WIP 2 - 4v4 Obstacle Course

A 4v4 PvP obstacle course where teams race through hazards while trying to disrupt each other. The main focus is network feel, fair collisions, and clearly readable hazard telegraphs.
